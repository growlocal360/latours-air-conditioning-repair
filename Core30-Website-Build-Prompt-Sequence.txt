   │   └── [sub-service]/index.html
   ├── css/styles.css
   ├── js/main.js
   └── images/

2. GLOBAL ELEMENTS:
   - Header with logo, navigation, phone CTA
   - Footer with NAP, service links, legal links
   - Mobile-responsive navigation
   - Consistent CTAs throughout

3. SCHEMA MARKUP (on every page):
   - LocalBusiness (consistent NAP)
   - BreadcrumbList
   - Service (on service pages)
   - AreaServed (on location pages)

4. SEO ELEMENTS:
   - Unique title tags (under 60 chars)
   - Meta descriptions (under 160 chars)
   - Canonical URLs
   - Open Graph tags
   - Mobile viewport meta

Please create the CSS framework and global HTML template structure.
